### Генератор Лидов ТГ

Админ панель для управления телеграм-юзер-ботами для генерации лидов по сделкам в реальном времени. Включает в себя человекоподобное поведение ботов и парсинг лидов по ключевым словам.

Длительность: 2 месяца

Стек:
`Python 3.11`
`Django 4.2`
`MySQL 8`
`Pyrogram 2`
`Telegram API`
`Docker SDK`
`Nginx`
`Ubuntu 24`
`Websockets`

Проектные решения:
- Проектирование базы данных
- Разработка админ панели для создания и управления телеграм-ботами
- Разработка админ-панели для авторизации телеграм-ботов
- Интеграция OpenAI API (ChatGPT) в телеграм-боты
- Разработка промптов для LLM
- Разработка системы человеческого поведения телеграм-ботов
- Разработка системы парсинга телеграм-групп для генерации лидов по ключевым словам
- Разработка системы автоматического перевода бота в ручной режим при ошибке системы или неудачного диалога
- Разработка системы контейнеризации телеграм-ботов с использованием Docker SDK
- Разработка системы мониторинга за контейнерами с телеграм-ботами в реальном времени, вшитую в админ панель
- Разработка системы управления телеграм-ботами с помощью команд в реальном времени, вшитую в админ панель
- Разработка чата внутри админ панели, позволяющего общаться с пользователем в телеграм от лица бота с помощью вебсокетов
- Разработка системы уведомлений менеджера о различных событиях
- Настройка сервера
- Настройка базы данных
- Автоматизация процессов деплоя с помощью github actions
- Настройка управления инфраструктурой
- Настройка NGINX
