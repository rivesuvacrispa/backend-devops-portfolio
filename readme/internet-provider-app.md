### Приложение Оператора Связи

Бекенд для мобильного приложения услуг оператора связи.

Длительность: 12 месяцев

Стек:
`PHP`
`Symfony`
`MySQL`
`REST`

Проектные решения:

- Внедрение нового функционала в существующую кодовую базу
- Написание логики верификации пользователя с помощью MTS KYC, SberbankAPI и AlfaBankAPI
- Документирование REST API
- Миграция данных
- Настройка серверного окружения
