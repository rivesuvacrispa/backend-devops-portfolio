### Онлайн-Кинотеатр

Веб платформа для израильского онлайн-кинотеатра.

Длительность: 4 месяца

Стек:
`PHP`
`Laravel`
`Filament`
`MySQL`
`REST`

Проектные решения:
- Проектирование базы данных
- Разработка интеграции со стриминговыми сервисами
- Разработка парсера новостного контента для блога
- Разработка REST API для фронта
- Документирование REST API
- Настройка серверного окружения
