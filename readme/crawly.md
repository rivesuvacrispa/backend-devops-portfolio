### Crawly Chronicles

Видеоигра в жанре action-survival с интересным твистом. Участник Brackeys GameJam 2023.1, и первая версия была написана с нуля за одну неделю.

Длительность: 2 месяца

Стек:
`Unity 2021`
`Unity 6`
`C# 8.0`

Проектные решения:
- Разработка идеи, соответствующей теме соревнования
- Разработка фундаментального концепта игры и основных её механик
- Разработка уникального визуального стиля
- Подбор и дизайн звуковых эффектов
- Подбор и дизайн визуальных эффектов
- Создание спрайтов и анимации
- Разработка игрового цикла
- Разработка интерфейса пользователя
- Разработка боевой системы, основанной на физике, с механиками парирования, брони, сопротивления
- Разработка системы дня и ночи
- Разработка системы сложности игры
- Разработка системы навыков персонажа с возможностью создать уникальную сборку для прохождения игрового процесса
- Разработка системы искусственного интеллекта противников, в которую входит поиск пути, стейты, сложные задачи, смены задач и различные уникальные поведения
- Разработка механик сильных противников-боссов, каждый из которых имеет уникальные боевые и поведенческие способности
- Обеспечение возможности выбора двух разных языков интерфейса
- Тестирование и исправление багов
