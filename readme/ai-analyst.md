### ИИ Анализатор Звонков

Веб-приложение для транскрибации и аналитики звонков. Интегрируется с Bitrix24 и AmoCRM с помощью вебхуков.

Длительность: 12 месяцев

Стек:
`Python 3.11`
`Django 4.2`
`PostgreSQL 15`
`OpenAI API`
`Bitrix24 REST API`
`AmoCRM API`
`Speech2Text API`
`Redis`
`Nginx`
`Docker`
`Debian 11`
`REST`

Проектные решения:
- Анализ и моделирование бизнес-процессов с помощью UML диаграмм
- Проектирование базы данных
- Докеризация приложения
- Разработка админ панели для управления клиентами и контентом
- Интеграция сервиса транскрибации речи
- Интегрирация OpenAI API (ChatGPT)
- Разработка промптов для LLM
- Разработка системы авторизации через JWT токен
- Разработка защищенного REST API для фронта
- Интеграция Bitrix24 через вебхуки для автоматической обработки входящих звонков менеджеров
- Интеграция AMO CRM через вебхуки и приватную интеграцию для автоматической обработки входящих звонков менеджеров
- Внедрение слоя кеширования данных
- Разработка асинхронного многопоточного сервиса для параллельной обработки звонков
- Разработка мониторинга генерации отчетов звонков в реальном времени с различными статусами (В процессе, Завершено, Ошибка)
- Разработка системы прав пользователей с различными правами на сущности для пользователей и групп пользователей
- Разработка внутренней системы валюты (минуты) 
- Документирование REST API 
- Настройка сервера
- Настройка базы данных
- Автоматизация процесса деплоя с помощью github actions
- Настройка и управление инфраструктурой
- Настройка NGINX
- Настройка DNS и HTTPS
- Настройка коллектора логов (Grafana, Loki)
- Настройка системы мониторинга (Grafana, Prometheus)
