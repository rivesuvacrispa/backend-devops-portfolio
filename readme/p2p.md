### P2P Крипто-Портал

Веб-приложение для управления P2P платформами с реферальными системами. Позволяет публиковать, модерировать платформы и выводить средства в TON.

Длительность: 12 месяцев

Стек:
`Python 3.11`
`Django 5`
`SQLite`
`Telegram API`
`Blockchain`
`REST`

Проектные решения:
- Проектирование базы данных
- Анализ и моделирование бизнес-процессов с помощью UML диаграмм
- Разработка системы ролей пользователя (Модератор, Платформа, Трейдер)
- Разработка REST API для фронта
- Документирование REST API
