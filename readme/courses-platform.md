### Платформа Медицинских Курсов

Платформа для покупки онлайн курсов с админ панелью для управления контентом.

Длительность: 4 месяца

Стек:
`PHP 8.1`
`Laravel 10`
`MySQL 8`
`Bitrix24 API`
`REST`

Проектные решения:
- Проектирование базы данных
- Разработка продвинутой интеграции с Bitrix24, включающую в себя синхронизацию каталога товаров, лидов, сделок, генерацию и поставку документов
- Интеграция платёжной системы AlfaBank
- Разработка REST API для фронта
- Документирование REST API
