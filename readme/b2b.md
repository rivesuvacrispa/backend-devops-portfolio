### B2B Кабинет

Бекенд для мобильного приложения услуг оператора связи.

Длительность: 12 месяцев

Стек:
`PHP`
`Symfony`
`MySQL`
`REST`

Проектные решения:
- Проектирование базы данных
- Разработка продвинутой интеграции с Bitrix24, включающую в себя синхронизацию каталога товаров, лидов, сделок, генерацию и поставку документов
- Разработка REST API для фронта
- Документирование REST API
