### Приложение Сети Химчисток

Бекенд для мобильного приложения крупной сети химчисток в Москве.

Длительность: 6 месяцев

Стек:
`PHP 8.4.1`
`Laravel 12`
`Filament 5`
`PostgreSQL`
`Teletrack API`
`Maxma API`
`AGBIS API`

Проектные решения:
- Анализ и моделирование бизнес-процессов
- Проектирование архитектуры
- Разработка базы данных
- Разработка всего функционала системы
- Интеграция CRM системы AGBIS для синхронизации клиентов и товарного каталога
- Интеграция курьерского сервиса Teletrack для автоматического создания курьерских задач на доставку
- Интеграция сервиса системы лояльности Maxma
- Разработка REST API
- Документирование REST API
- Настройка сервера
- Настройка DNS, HTTPS и NGINX
- Автоматизация процесса деплоя с помощью gitlab action
