### Редактор Электрических Сетей

Платформа для создания карт городских электрических сетей.

Длительность: 12 месяцев

Стек:
`PHP 8.2`
`Laravel 11`
`MySQL 8`
`Redis`
`Websockets`

Проектные решения:
- Внедрение нового функционала в существующую кодовую базу
- Исправление багов
- Помощь в локальном развертывании проекта новым членам команды
